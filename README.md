# Crypto_Price_Prediction

This is the crpyto currency buy/sell predictor, where the custom CNN model is trained to predicted whether to buy or sell the crypto. 

The crypto currencies choosen here are:
1. "BCH-USD"
2. "BTC-USD"
3. "ETH-USD"
4. "LTC-USD"
